🔧 איך פותרים הכל — צעד אחר צעד

✅ האתר כבר באוויר ב-ilanmichalby.github.io/shabbat-yerucham (מתפרסם מתיקיית docs/). הצעד הבא הכי משתלם: בניית AAB והעלאה ל-Google Play.

1️⃣ times.html ✅ הושלם

1

פתח את .stitch/next-prompt.md

הפרומפט המלא לדור העמוד כבר שמור שם — עם כל מערכת העיצוב.

2

תן ל-Claude לבנות

הדבק את הפרומפט ל-Claude (או Stitch) וציין: "צור times.html בסגנון index.html הקיים, עם JavaScript שמושך shabbat-times.json".

3

שמור ב-docs/times.html

הגדרת הנב' ב-index.html כבר מקשרת לשם — לא צריך לשנות כלום.

2️⃣ בניית APK לאנדרואיד — הפצה ישירה (ללא חנות)

1

התקן EAS CLI

npm install -g eas-cli ו-eas login עם חשבון Expo של ilanb

2

בנה APK להפצה פנימית

מתוך תיקיית mobile/: eas build -p android --profile preview

3

הורד ושתף

EAS ישלח קישור להורדת APK — שלח ישירות לכל מי שרוצה להתקין.

3️⃣ העלאה ל-Google Play — חתימה כבר מוכנה

🔑 חשוב: קבצי החתימה (.jks, upload_certificate.pem) נמצאים בתיקיית mobile/ מחוץ לגיט — לגבות אותם במקום בטוח! בלעדיהם אי אפשר לעדכן את האפליקציה בחנות.
1

בנה AAB (Android App Bundle)

מתוך תיקיית mobile/: eas build -p android --profile production — EAS יבנה בענן וישלח קישור.

2

צור אפליקציה ב-Play Console

play.google.com/console → "צור אפליקציה" → שם: זמני שבת ירוחם, שפה: עברית, קטגוריה: Lifestyle.

3

העלה ל-Internal Testing קודם

Testing → Internal testing → העלה את ה-AAB. בדוק שהכל עובד לפני שעוברים ל-Production.

4

מלא פרטי חנות

תיאור קצר + מלא בעברית, לפחות 2 צילומי מסך (Phone), Feature graphic (1024×500px), אייקון 512×512.

5

פרסם ל-Production

לאחר אישור Google (בד"כ 1–3 ימים) האפליקציה תהיה זמינה לכולם.

4️⃣ עדכון נתוני שבת (תחזוקה שוטפת)

1

הורד זמנים מ-HebCal

hebcal.com → Candle Lighting Times → עיר: Yerucham → ייצא CSV

2

פתח את לוח הניהול

docs/admin/index.html — או ישירות באוויר: ilanmichalby.github.io/shabbat-yerucham/admin/

3

העלה CSV ושמור

הלוח יייצר קובץ shabbat-times.json חדש — שמור ב-docs/ ועשה git push.

5️⃣ בניית about.html ו-parasha.html — בעתיד

1

about.html

הסבר: ירוחם ממוקמת בנגב (31.52°N), כניסת שבת = 18 דקות לפני שקיעה (מנהג ירוחם), צאת שבת = 42 דקות אחרי שקיעה, רבנו תם = 72 דקות.

2

parasha.html

ניתן להשתמש ב-API של HebCal בחינם: https://www.hebcal.com/shabbat?cfg=json&geo=city&city=Yeruham — מחזיר את הפרשה אוטומטית.
